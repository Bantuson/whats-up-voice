# Phase 06.1 Plan 02: Realtime Language Translation Summary

Session-based bidirectional translation using Claude + ElevenLabs multilingual TTS for SA languages (Zulu, Xhosa, Sesotho, Afrikaans, English).

## What Was Built

### Task 1: translation.ts tool + session machine translating state (TDD)

Extended `src/session/machine.ts`:
- Added `'translating'` to `SessionPhase` union
- Added `translationTarget?: string` and `detectedLanguage?: string` to `SessionState`
- Extended `TRANSITIONS`: translating reachable from idle/listening/playing, loops to itself
- Exported `setTranslationTarget`, `clearTranslationTarget`, `setDetectedLanguage`

Created `src/tools/translation.ts`:
- `LANGUAGE_NAMES` map — BCP-47 codes to full names for Claude prompts (zu→Zulu, xh→Xhosa, st→Sesotho, af→Afrikaans, en→English, fr→French, pt→Portuguese, sw→Swahili)
- `activateTranslation(userId, targetLanguage)` — transitions session to translating, stores target, returns spoken confirmation with full language name
- `deactivateTranslation(userId)` — clears translation target, returns to idle
- `translateUtterance(userId, text)` — calls Claude with language-aware system prompt, delivers via streamSpeechInLanguage

Extended `src/tts/elevenlabs.ts` (done in Task 1 to unblock tests):
- `VOICE_ID_MAP` for all SA languages: af/zu/xh/st/en
- `selectModel` now returns `eleven_multilingual_v2` for all non-English (was 'af'-only)
- New export `streamSpeechInLanguage(text, userId, languageCode)` — bypasses Supabase profile lookup

### Task 2: Classifier, orchestrator, api.ts wiring

Updated `src/agent/classifier.ts`:
- Added `'start_translation'` and `'stop_translation'` to `FastPathIntent` union
- Added two FAST_PATH patterns before search/ambient queries

Updated `src/agent/orchestrator.ts`:
- Imported `activateTranslation`, `deactivateTranslation`, `translateUtterance`
- Added `ActivateTranslation`, `DeactivateTranslation`, `TranslateUtterance` to ALL_TOOLS (now 13 tools)
- Added 3 cases to executeTool switch

Updated `src/routes/api.ts`:
- Imported `activateTranslation`, `deactivateTranslation`, `translateUtterance`, `setDetectedLanguage`
- Added Whisper language detection storage: `setDetectedLanguage(userId, result.language)` on every STT request
- Added `stop_translation` fast-path handler
- Added `start_translation` fast-path handler with language name extraction and code mapping
- Added translation mode intercept: when `session.phase === 'translating'` and `intent === null`, routes to `translateUtterance`
