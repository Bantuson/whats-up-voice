# Phase 06.1 Plan 01: Generated Podcasts Feature Summary

**One-liner:** VI users can say "tell me something about Kaizer Chiefs" and receive a 2-5 minute natural-speech podcast via WhatsApp voice note, synthesised from Tavily research through Claude.

## What Was Built

The Generated Podcasts feature (VI-PODCAST-01, VI-PODCAST-02, VI-PODCAST-03) enables VI users to request personalised audio content on any topic by voice. The pipeline:

1. Fast-path classifier matches `podcast_request` or `short_version` intent without LLM invocation
2. `generatePodcast(topic, userId, shortVersion?)` in `src/tools/podcast.ts`:
   - Researches topic via Tavily (searchDepth: 'advanced', maxResults: 5)
   - Synthesises natural-speech script via Claude (PODCAST_SYSTEM_PROMPT enforces zero markdown)
   - Runs `sanitiseForSpeech()` as post-processor defence
   - Streams result via ElevenLabs TTS to user's WebSocket
3. Fast-path in `api.ts` dispatches `podcast_request` directly; `short_version` routes through orchestrator for conversation context
4. `GeneratePodcast` tool registered in `ALL_TOOLS` for LLM-path invocation
